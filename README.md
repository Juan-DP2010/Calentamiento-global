# Calentamiento global
En este proyecto, yo y mi subconciente buscamos concientizar acerca del calentamiento global, causas, consecuencias, soluciones y preguntas para debates acerca del mismo.

# Primero que todo: ¿Qué es el calentamiento global?
Es el aumento progresivo de la temperatura promedio de la Tierra gracias a gases de efecto invernadero en la atmósfera.
